Packages for Debian / Ubuntu

Aptly packages are built by the github pipeline for various Debian / Ubuntu distributions and architectures.

Importing GPG keys

Download and install repo.aptly.ingo GPG keys:

$ sudo wget -O /etc/apt/keyrings/aptly.asc https://www.aptly.info/pubkey.txt

For releases older than Debian 12 and Ubuntu 22.04, create the directory first:

$ sudo mkdir -p /etc/apt/keyrings; sudo chmod 755 /etc/apt/keyrings

Adding APT Source

Create new file /etc/apt/sources.list.d/aptly.list with: 

deb [signed-by=/etc/apt/keyrings/aptly.asc] http://repo.aptly.info/release DIST main

Where DIST is one of:

Debian: bullseye / bookworm / trixie

Ubuntu: focal / jammy / noble / resolute

For testing CI Builds built from master branch, replace release with ci in the URL.

Install Aptly

$ sudo apt-get update
$ sudo apt-get install aptly  # aptly cli
$ sudo apt-get install aptly-api  # API server, systemd service

Official Debian / Ubuntu Packages

Aptly packageis are available in Debian as well as Ubuntu.

These packages come with bug fixes but do not offer all features, depending on the availability of go dependencies.

Mac OS X

Mac OS X: Installation using Homebrew:

$ brew install aptly
