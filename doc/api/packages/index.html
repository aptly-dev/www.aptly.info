Package API

APIs related to packages on their own.

Show

GET /api/packages/:key

Show information about package by package key.

Package keys could be obtained from various GET .../packages APIs.

Response:

NameTypeDescription
Keystringpackage key (unique package identifier)
ShortKeystringshort package key (should be unique in one package list: snapshot, mirror, local repository)
FilesHashstringhash of package files
Package Stanza Fieldsstringall package stanza fields, e.g. Package, Architecture, …

HTTP Errors:

CodeDescription
404package with such key doesn’t exist

Example:
